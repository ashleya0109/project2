Project 2

Amazon is widely used across the United States however there isnt great data surrounding price fluctuations and best seller rankings. This project focused on collecting data to see an items specific trend over a week

Data Collecting Technique: Data was collected by screen scraping multiple amazon web pages. I determined what categories I wanted to screen scrape and added to an Array in python along with the base websites that contained the data I needed. This was a two part approach 1st collected high level details about the products but to retrieve the keywords associated with each item I needed to use data from the 1st screen scrape to do a second. Data was stored in a SQL database on Postgres.
Cleaning Data: After collecting all the data needed I begin to use sql in python to manipulate the data into various csv that I would need to access via javascript
Technology Used: Python, SQL Database, Beautiful soup, Javascript
